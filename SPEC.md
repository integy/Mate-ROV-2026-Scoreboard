# MATE ROV 2026 Explorer Class 計分板

## 項目概述
- **名稱:** MATE ROV 2026 Explorer Class Scoreboard
- **類型:** Web App (單頁應用)
- **用途:** MATE ROV Competition 2026 Regional 計分
- **主題:** "Pushing Performance: Science, Technology, Discovery In Harsh Environments"

## 總分結構

| 項目 | 最高分數 |
|------|----------|
| Safety (安全) | 5 分 |
| Organizational and Operational Effectiveness | 10 分 |
| Weight Bonus (重量獎勵) | 10 分 |
| Task 1: Seabed 2030 | 85 分 |
| Task 2: SmartAtlantic | 135 分 |
| Task 3: Wind-Powered Offshore Oil Platform | 60 分 |
| Task 4: MATE Floats Under the Ice | 70 分 |
| **總分** | **~375 分** |

## UI/UX 設計

### 配色方案
- 深色主題配合海底氛圍
- 主色: #0a1628 (深海藍黑)
- 卡片背景: #16213e
- 重點色: #00d4ff (水下藍光)
- 紅隊: #e63946
- 藍隊: #457b9d
- 成功: #2a9d8f
- 警告: #f4a261

### 佈局
- 頂部: 團隊名稱輸入 + 總分顯示
- 主要區域: 4個任務標籤頁面
- 側邊欄: 當前任務摘要
- 底部: 快捷操作 (重置、匯出、匯入)

### 功能

#### 1. 團隊管理
- 輸入公司/團隊名稱
- 輸入 CEO 名字
- 輸入裁判名字

#### 2. Safety (5分)
- ✅ 安全檢查表通過 (需要勾選)
- ✅ JSEA 簡報完成 (需要勾選)
- 顯示: 0-5 分

#### 3. Organizational Effectiveness (10分)
- Teamwork/Collaboration (4分)
  - 指揮鏈明顯 (0-1)
  - 角色清晰且執行良好 (0-1)
  - 壓力下保持鎮定 (0-1)
  - 互相支持且良好溝通 (0-1)
- Project Management (2分)
  - 任務計劃周詳 (0-1)
  - 有備用計劃 (0-1)
- Problem Solving (2分)
  - 時間意識 (0-1)
  - 調整時間線適應情況 (0-1)
- System Design (2分)
  - 5分鐘內部署 ROV (0-1)
  - ROV 由控制員控制並準備好執行任務 (0-1)

#### 4. Weight Bonus (10分)
- < 18 kg = 10 分
- 18.01-25 kg = 5 分
- 25.01-35 kg = 0 分
- > 35 kg = 不能參賽

#### 5. Task 1: Seabed 2030 (85分)
- 1.1 收集物種 (30分)
  - 收集 2 個籃星 (10分)
  - 收集 2 種珊瑚 (20分)
  - 從基底收集 vs 非基底收集
- 1.2 珊瑚嶺建模 (40分)
  - 攝影測量自主創建 3D 模型 (20分)
  - 手動 CAD 創建 3D 模型 (10分)
  - 測量長度 (10分)
  - 測量高度 (5分)
  - 縮放模型 (5分)
- 1.3 飛越海床測線 (15分)

#### 6. Task 2: SmartAtlantic (135分)
- 2.1 清除入侵物種 (20分)
  - 識別青蟹數量 (15分)
  - 上傳數據 (5分)
- 2.2 冰山追蹤 (35分)
  - 測量 5 點周邊 (10分)
  - 測量龍骨深度 (10分)
  - 確定對 4 個油氣平台既威脅級別 (15分)
- 2.3 鯨魚安全捕魚設備 (20分)
  - 轉動把手釋放浮標 (10分)
  - 回收龍蝦罐 (10分)
- 2.4 回收浮標錨 (15分)
- 2.5 服務 Holyrood 海底觀測站 (45分)
  - 回收舊 eDNA 感測器 (15分)
  - 分析數據 (10分)
  - 安裝新 eDNA 感測器 (20分)

#### 7. Task 3: Wind-Powered Oil Platform (60分)
- 3.1 微樁安裝 (30分)
  - 放置氣泡幕裝置 (10分)
  - 引導微樁到指定位置 (10分)
  - 拉開固定針釋放微樁 (10分)
- 3.2 為油氣平台供電 (30分)
  - 回收 power connector (5分)
  - 鋪設電纜通過路點 (10分)
  - 移除平台蓋 (5分)
  - 安裝 connector (10分)

#### 8. Task 4: Floats Under Ice (70分)
- 設計並製作浮筒 (5分)
- 與站點通訊 (5分)
- 第一次潛水剖面 (25分)
  - 使用浮力引擎完成 (10分)
  - 維持 2.5m 深度 30秒 (5分)
  - 維持 40cm 深度 30秒 (5分)
  - 唔好撞冰面 (-5分)
- 第二次潛水剖面 (25分)
- 恢復後通訊 (10分)
- 圖表化數據 (10分)

## 技術實現

- 單一 HTML 檔案 (HTML + CSS + JS 內聯)
- LocalStorage 儲存數據
- 匯出/匯入 JSON
- 列印友好
- 響應式設計 (手機/平板/電腦)

## 驗收標準

1. ✅ 所有分數項目都可以輸入
2. ✅ 總分自動計算
3. ✅ 數據可以儲存同匯出
4. ✅ 界面美觀，符合水下主題
5. ✅ 支援列印輸出
